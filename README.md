A simple multi-page website built using semantic HTML.

This project is part of the Frontend Developer roadmap from roadmap.sh.

## Features

-   Semantic HTML structure
-   Multi-page website
-   Navigation between pages
-   SEO meta tags
-   Contact form
-   Accessible markup

## Tech Stack

-   HTML5
